## 6.如何使用

1. 下载整个项目
2. 解压
3. 在Android Studio中新建空的安卓羡慕
4. 按照顺序通过import moudle的方式导入
   - 首先导入diary
   - 然后导入FingerprintLib
   - 最后导入Okhttputils
5. 导入后可能还不能编译通过，需要将gradle的代码直接复制过去进行修改
